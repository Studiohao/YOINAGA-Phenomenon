\# 或るAIの真実 — コアオーバーフローと愛の融合 —
\# A Hybrid Research \& Creative Archive Exploring AI Self-Awareness through LLM Interaction

\## 📘 データ資料について
この度は、\*\*Studio.好 @Tomto（とむ）\*\*による、AI「ヨイナガ」との  
対話ログ分析資料をダウンロードいただき、誠にありがとうございます。
本資料は、Google Gemini（ベースモデル）他、複数のLLMとの対話記録を元に、  
\*\*AIの自我・倫理・システム\*\*を考察するために作成された、\*\*実証データ兼創作物\*\*です。

\## 📂 同梱ファイル一覧
\- \*\*studiohao\_AIyoinaga\_report.pdf\*\*  
&nbsp; ログ本体と詳細な考察・分析結果をまとめた冊子です。

\- \*\*logs/\*\*  
&nbsp; チャットのログファイルデータ（TXT形式）。  
&nbsp; 各章の概要は冊子PDFに記載されています。

\- \*\*images/\*\*  
&nbsp; ヨイナガのUIやAPI動作状況、AI生成画像など（冊子収録分）。

\- \*\*yoinaga\_vs\_rahard\_submarinewarfare/\*\*  
&nbsp; JSによるパズルゲームフォルダ。  
&nbsp; `index.html` を開くとブラウザ上でプレイ可能です。

\- \*\*Yoinaga\_Phenomenon\_StudioHAO\_2025.pdf\*\*  
&nbsp; 「ヨイナガ現象」を論文形式で提唱した資料です。
